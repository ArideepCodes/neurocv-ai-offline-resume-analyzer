# NeuroCV AI - Offline AI Resume Analyzer

An advanced futuristic AI-powered ATS Resume Analyzer built entirely inside a single `index.html` file using pure frontend technologies.

Developed by Arideep Kanshabanik

---

## Overview

NeuroCV AI is a fully offline browser-based resume intelligence platform designed to simulate enterprise-grade AI recruitment analysis without using any backend or external APIs.

The project delivers a premium SaaS-like experience with futuristic cyberpunk UI, advanced animations, interactive analytics, ATS scoring simulation, AI-style recommendations, and modern dashboard architecture.

Everything runs locally inside the browser.
